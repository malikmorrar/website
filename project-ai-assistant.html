Overview

The Problem

Businesses handling high inbound call volume face a real bottleneck: every missed or slow-handled call is a lost customer. Manual call handling is inconsistent, slow, and doesn't scale. The goal was to build a system that could receive, understand, and respond to customer calls automatically — 24/7, without staff involvement.

The Solution

I built a fully automated AI voice assistant using Twilio Voice API as the telephony layer. When a call comes in, Twilio routes it via webhook to a custom backend that processes the audio in real time using speech-to-text. The transcript is passed to the OpenAI API, which classifies the caller's intent and determines the appropriate response flow. The assistant replies via text-to-speech and can handle bookings, answer FAQs, and route edge cases through fallback logic. Every interaction is transcribed, structured, and automatically pushed to HubSpot CRM via Zapier — creating a full audit trail with zero manual entry.
